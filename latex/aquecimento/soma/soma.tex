\problema{(Soma)}

\begin{descricao}
Dada uma lista de pares de inteiros, você deve calcular a soma de cada par.
\end{descricao}

\begin{entrada}
A entrada é composta por diversos casos de teste, um por linha. Em cada linha, há dois inteiros, $1 \leq a, b \leq 1000$. A última linha contém $a=b=0$ e não deve ser processada.
\end{entrada}

\begin{saida}
Para cada caso de teste, deve ser impresso o valor de $a+b$.
\end{saida}

\exemplos{0.47}{0.47}
{10 1	\pl
400 44	\pl
1 1	\pl
0
}
{11	\pl
444	\pl
2
}
